🔧 Configuración adicional:
==========================

1. Configurar DNS local (/etc/hosts):
   # BlinkChamber Services
   127.0.0.1 vault.{{ .Values.global.domain }}
   127.0.0.1 identity.{{ .Values.global.domain }}
   127.0.0.1 mail.{{ .Values.global.domain }}
   127.0.0.1 monitoring.{{ .Values.global.domain }}

2. Verificar estado del release:
   helm status {{ .Release.Name }} -n {{ .Release.Namespace }}

3. Ver logs de componentes:
   kubectl logs deployment/{{ include "blinkchamber.fullname" . }}-vault -n {{ .Release.Namespace }}

4. Configurar port-forwards automáticamente:
   ./scripts/blinkchamber-helm.sh port-forward --all
